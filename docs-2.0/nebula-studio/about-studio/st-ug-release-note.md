# Studio版本更新说明
## v2.2.1（2021.05.06）

- 修复
  - 修复浏览器兼容问题
  - 修复 big int 类型数据渲染问题
  - 修复导入数据无法获取结果的问题

## v2.2.0 (2021.04.23)

- 功能增强:
  - 图探索
    - UI 改版
    - 新增操作面板，右键菜单，支持缩放、拓展、修改节点颜色等操作
    - 支持部分操作快捷键操作
    - 支持快速查找画板中相关节点
    - 拓展功能增强，支持多步查询
    - 新增图算法查询功能

- 修复: 
  - 控制台
    - 修复 MATCH 返回结构无法导向图探索的问题
