# 版本说明

不同版本的生态工具支持的Nebula Graph内核版本不同，本文介绍生态工具版本和Nebula Graph内核版本的对应关系。

!!! Note

    1.x版本的生态工具不支持在Nebula Graph 2.x版本中使用。
