# SHOW EDGES

`SHOW EDGES`语句显示当前图空间内的所有边类型名称。

执行`SHOW EDGES`语句不需要任何权限，但是返回结果由登录的用户[权限](../../7.data-security/1.authentication/3.role-list.md)决定。

## 语法

```ngql
SHOW EDGES;
```

## 示例

```ngql
nebula> SHOW EDGES;
+----------+
| Name     |
+----------+
| "follow" |
+----------+
| "serve"  |
+----------+
```
