# SHOW INDEXES

`SHOW INDEXES`语句可以列出当前图空间内的所有Tag和Edge type（包括属性）的索引。

## 语法

```ngql
SHOW {TAG | EDGE} INDEXES;
```

## 示例

```ngql
nebula> SHOW TAG INDEXES;
+------------------+
| Names            |
+------------------+
| "fix"            |
+------------------+
| "player_index_0" |
+------------------+
| "player_index_1" |
+------------------+
| "var"            |
+------------------+

nebula> SHOW EDGE INDEXES;
+------------------+
| Names            |
+------------------+
| "follow_index_0" |
+------------------+
```
