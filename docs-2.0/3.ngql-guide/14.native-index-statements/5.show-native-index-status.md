# SHOW INDEX STATUS

`SHOW INDEX STATUS`语句可以查看索引名称和对应的状态。

索引状态包括：

- `QUEUE`：队列中
- `RUNNING`：执行中
- `FINISHED`：已完成
- `FAILED`：失败
- `STOPPED`：停止
- `INVALID`：失效

!!! Note

    如何创建索引请参见[CREATE INDEX](1.create-native-index.md)。

## 语法

```ngql
SHOW {TAG | EDGE} INDEX STATUS;
```

## 示例

```ngql
nebula> SHOW TAG INDEX STATUS;
+----------------------+--------------+
| Name                 | Index Status |
+----------------------+--------------+
| "player_index_0"     | "FINISHED"   |
+----------------------+--------------+
| "player_index_1"     | "FINISHED"   |
+----------------------+--------------+
```
