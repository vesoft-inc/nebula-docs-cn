# SHOW HOSTS

`SHOW HOSTS`语句显示由Meta服务注册的graph、storage、meta主机。

## 语法

```ngql
SHOW HOSTS [GRAPH/STORAGE/META];
```

## 示例

```ngql
nebula> SHOW HOSTS;
```
