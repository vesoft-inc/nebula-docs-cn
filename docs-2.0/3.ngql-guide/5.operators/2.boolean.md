# 布尔符

Nebula Graph支持的布尔符如下。

| 符号 | 说明 |
| :------- | :-------------: |
| AND      |   逻辑与   |
| OR       |   逻辑或    |
| NOT      |   逻辑非   |
| XOR      |   逻辑异或   |

对于以上运算的优先级，请参见[运算优先级](9.precedence.md)。

对于带有`NULL`的逻辑运算，请参见[NULL](../3.data-types/5.null.md)。

## 遗留问题

在Nebula Graph 2.0中，非0数字不能转换为布尔值。
